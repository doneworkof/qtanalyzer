Внимание!
Для запуска проекта, необходимо
запустить файл loader.py:

python loader.py

Файлы проекта:
1) loader.py -- форма загрузки БД
2) body.py -- форма для работы с БД
3) tools.py -- необходимые для проекта мелкие функции и константы
4) analysis_form.py -- форма, показывающая анализ
5) analysis_center.py -- форма для обобщения разных
  анализов для одной колонки
6) analysis.py -- самый главный файл решения, ядро всех вычислений
7) about_project.docx -- описание проекта
8) requirements.txt -- необходимые библиотеки для Python для корректной работы приложения
